Welcome!

My name is Helge Holzmann. I’m a computer scientist, web archiving and data expert, and a web data enthusiast with a strong interest in Big Data, especially when there is a temporal aspect to it. I hold a PhD in Computer Science, where my research focused on efficient access methods for web archives.

Over the past years, I have worked as a Web Data Engineer with the Internet Archive, the non-profit organization behind one of the largest collections of freely accessible data in the world. Building on this experience, I continue to develop innovative services and help derive new value from large-scale web archive datasets. As an independent consultant based in Europe, I also support libraries and institutions interested in accessing and working with (temporal) web data at scale.

On this website you can find information about:

  • Consulting services
  • My publications
  • Talks and presentations
  • Software projects

If you would like to learn more or discuss potential collaboration, please feel free to get in touch.
