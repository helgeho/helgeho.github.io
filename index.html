Welcome!

My name is Helge Holzmann, I’m a computer scientist, web archiving and data expert and passionate about Big Data, especially if there’s a temporal aspect to it. I have a PhD, which I wrote on efficient access methods for web archives, and work as Web Data Engineer for the Internet Archive, a non-profit organization that holds one of the biggest collections of free data in the world. In addition to creating innovative services by deriving new value from this unique dataset, as a consultant located in Europe, I’m happy to support libraries and institutions in accessing (temporal) web data at scale.

On this website you can find information on the following topics:

  • My publications

  • Software projects

For more, please get in touch.
